NeoBoard

Uma aplicação web desenvolvida em grupo para o monitoramento de empresas de vendas, que possuem diversos vendedores e produtos. A plataforma permite o cadastro de vendedores e produtos, exibindo os dados de cada um de forma organizada. Além disso, a aplicação gera gráficos de desempenho de vendas, proporcionando uma visão clara e intuitiva do desempenho individual e geral, facilitando a análise e a tomada de decisões estratégicas. Ver Projeto

NeoWeather

Um aplicativo móvel desenvolvido para exibir as condições climáticas atuais e a previsão do tempo para os dias seguintes em qualquer cidade do mundo. A aplicação faz uso de uma API de previsão do tempo para obter dados precisos e em tempo real. O usuário pode inserir o nome de qualquer cidade e receber informações, como temperatura condições gerais do clima, proporcionando uma experiência intuitiva e útil para planejamentos futuros.Ver Projeto
